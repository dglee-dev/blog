# 구현 마일스톤

## Milestone 1. Vercel KV 연동 및 Lambda 수정

Lambda에 Vercel KV를 연동하고, 요청 시 KV에서 캐시된 목록을 반환하도록 수정한다. S3 목록과 length 비교 후 다르면 KV를 업데이트한다.

**체크리스트**

- [ ] Vercel KV 프로젝트 생성 및 환경변수 설정
- [ ] Lambda에서 KV 읽기/쓰기 동작
- [ ] S3 목록과 length 비교 후 KV 업데이트
- [ ] Lambda 응답에 캐시 목록 포함

---

## Milestone 2. 클라이언트 스켈레톤 구현

Lambda 응답의 캐시 목록으로 스켈레톤을 즉시 렌더링하고, 실제 데이터 도착 시 교체한다.

**체크리스트**

- [ ] 캐시 목록 개수만큼 스켈레톤 아이템 렌더링
- [ ] 실제 데이터 도착 시 스켈레톤 → 실제 리스트 교체
- [ ] 첫 방문 시 스켈레톤 즉시 표시
- [ ] prefetch 캐시 있을 때 스켈레톤 없이 즉시 렌더링
